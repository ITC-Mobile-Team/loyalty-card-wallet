# Design Image References

Use this folder for screenshots, mockups, visual references, and design research assets.

## Naming

Use descriptive names:

```text
cards-grid-reference-ios.png
add-card-flow-reference.png
barcode-scanning-mode-sketch.png
```

## Rules

- Do not store private personal card numbers in screenshots.
- Prefer cropped images that show only the relevant UI.
- Add short notes in `docs/research/` when an image affects a design decision.
