# API And Contract Notes

The MVP has no backend API.

This directory is reserved for contracts that still matter in a local-first app:

- import/export JSON format,
- local repository interfaces,
- future catalog update format,
- future sync API notes,
- permission and platform adapter contracts.

## Current Contracts

- [Import and export contract](import-export.md)

The first export format should include cards, pictures, catalog overrides when present, and enough metadata to detect incompatible future versions.
